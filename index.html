🔄 How to Auto-Populate This Data
Google Analytics 4
1. In your Google Sheet, click Extensions → Add-ons → Get add-ons
2. Search for "Google Analytics" (by Google LLC) and install it
3. Go to Extensions → Google Analytics → Create new report
4. Name it GA_Analytics and pick your GA4 property
5. Add metrics: Active Users, Conversions — dimensions: Page Path, Session Source
6. Set schedule to Run reports automatically (daily or weekly)
LinkedIn Organic
1. Go to coupler.io and create a free account
2. Create a new importer: LinkedIn Pages → Google Sheets
3. Connect your LinkedIn Company Page and your Google Sheet
4. Set the destination tab to LI_Organic
5. Map columns: Period, Impressions, Engagements, Followers, Follower_Change
6. Set schedule to weekly — Coupler runs it automatically
Sheet tab names must match exactly: GA_Analytics and LI_Organic — the dashboard reads these automatically.
